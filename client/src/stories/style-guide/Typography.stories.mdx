# Typography

## Font

We use [Poppins](https://fonts.google.com/specimen/Poppins) for all fonts.

The base font size is `16px`.

## Overview

<Overview />

### Header text

Used in the titles in a `<Content />` card.

Use `classes.headerText` to apply styles.

The size of this text can be `2.5rem`, `2rem`, or `1.35rem` depending on the screen width. The breakpoints are based on when the longest title of a card will start to word wrap. This lets us keep the titles to fit in one line.

### Left-aligned text

Used for the sub-header of a `<Content />` card.

### Centered text

Used in the instructions and status information in a card

### File card

Used in the `<FileLabel />` component.

### Primary button

The primary call-to-action (CTA) button. Text inside these buttons should be in Sentence case.

### Inactive primary button

Text inside these buttons should be in Sentence case.

### Secondary button

Text inside these buttons should be in Sentence case.

### Navigation button

Text inside these buttons should be in UPPERCASE.

### Form field [code]

Used in the code input
