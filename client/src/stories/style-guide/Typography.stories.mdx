## Notes

- We don't follow mantine's color system and instead store a custom color pallete. To use grey text, use `classes.grey` instead of the `color` prop.

```tsx
// right
// - Note: `classes` come from the useCommonStyles hook
<Text className={classes.grey} />

// wrong
<Text color="gray" />
<Text color="dimmed" />
```

- Similarly, use `classes.darkGrey` for darker text, and `classes.dark` for black text.
